# GranuFactory

A full-stack e-commerce solution developed for my girlfriend's small cake business named "Granu Factory" to manage orders, inventory, and payments online. 
The application was designed to streamline the ordering process, reduce manual administrative work, and provide a seamless user experience for customers. 
## What Technologies will I been using?
Frontend: React/Vite
Backend: Java/Springboot, JWT (for authorization)
Database: OracleCloud
